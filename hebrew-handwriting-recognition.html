About the Project

This project focuses on recognizing Hebrew cursive letters using deep learning models and transfer learning techniques. It includes:

  • Classification of 27 Hebrew cursive letters.
  • A dataset of 5,073 images.
  • Comparison between traditional CNN models and pre-trained models such as ResNet50 and VGG16.

Models and Techniques

We utilized two main approaches for classification:

CNN (Convolutional Neural Network)

  • Implemented regularization techniques (L1, L2).
  • Applied dropout layers and data augmentation to prevent overfitting.
  • Achieved an accuracy of 67% using early stopping and augmentation.

Transfer Learning

Pre-trained models were fine-tuned to improve performance:

  • ResNet50: Achieved 80% accuracy with fine-tuning (learning rate: 0.001, optimizer: Adam).
  • VGG16: Achieved 82% accuracy with fine-tuning.

Results

The best-performing model was VGG16 with fine-tuning, achieving an accuracy of 82%. Other observations include:

  • Transfer learning outperformed traditional CNN models.
  • Fine-tuning pre-trained models significantly improved accuracy.

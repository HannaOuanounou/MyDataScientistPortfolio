Automated Data Cleaning with ML

This project automates data cleaning using machine learning algorithms to improve data quality. The pipeline handles missing values, outliers, and inconsistent formatting. It includes advanced ML models like Isolation Forest for anomaly detection and supervised learning for imputing missing values.

Features:

  • Anomaly detection using .
  • Missing value prediction with machine learning models.
  • Automated data normalization and feature scaling.
  • Comprehensive data quality report generation.
